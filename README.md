# Blood Ascend Discord Bot

Bot Discord en Node.js (discord.js) avec une commande slash `ping`.

## Prérequis
- Node.js 18+
- Un bot Discord créé dans le portail développeur

## Installation
1. Installer les dépendances :
   - `npm install`
2. Copier `.env.example` vers `.env` et compléter :
   - `DISCORD_TOKEN`
   - `CLIENT_ID`
   - (optionnel) `GUILD_ID` pour enregistrer les commandes plus vite

## Démarrer le bot
- `npm start`

## Commandes
- `/ping` → répond `Pong!`
